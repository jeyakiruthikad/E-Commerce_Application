# UrbanShop - Simple E-Commerce Website

A clean and beginner-friendly **E-Commerce web application** built using **HTML, CSS, and JavaScript**.
This project simulates a basic online shopping platform with multiple sections like products, orders, wishlist, login, and reviews.

---

## Features

* Multi-page navigation (Home, About, Products, Orders, Wishlist, etc.)
* Product listing with images, price, and ratings
* Add to Cart functionality (alert-based)
* Wishlist section
* Customer reviews display and submission form
* User Login and Registration form with validation
* Contact / Customer support section
* Clean and responsive UI design

---

## Technologies Used

* HTML
* CSS (Flexbox, Layout, UI design)
* JavaScript (DOM manipulation, Event handling)

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/jeyakiruthikad/E-Commerce_Application.git
```

2. Navigate into the folder:

```bash
cd E-Commerce_Application
```

3. Open in browser:

```bash
start E-Commerce.html
```

## How It Works

* Navigation bar switches between sections using JavaScript
* Products can be added to cart (simulated with alerts)
* Forms (Login, Register, Review) use validation and event listeners
* Reviews can be submitted via form (no backend storage)
* All pages are handled in a single HTML file using dynamic display
